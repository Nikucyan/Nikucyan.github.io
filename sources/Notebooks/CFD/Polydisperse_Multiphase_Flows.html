              1. Introduction, NDF

              Definitions

              • Multiphase: Composited of various phases, specifically, 1 phase cont and other phases disperse
              • Disperse: One or more phases are composed of clearly identifiable discrete entities (elem): particles (solid) / droplets (liquid) / bubbles (gas)
              • Polydisperse: Properties of each entity can be different (<=> Monodisperse for same properties, e.g., same size particles …)

              Industrial applications: spray, bubble tower, etc.

              Coordinates

              • External: spatial (physical space)

                x≡(x1,x2,x3);dx≡dx1 dx2 dx3

              • Internal: the properties of the dispersed entities (phase space)

                ξ≡(ξ1,ξ2,…,ξm);dξ=dξ1 dξ2…dξm

                m internal coord of interest

              Number Density Function (NDF)

              Similar to probability density function (PDF) in statistics

              nξ(t,x,ξ)

              nξ dx dξ: expected # of entities in dx & dξ (to solve nξ => describe the dispersed entity distributions)

              0th Order Moment

              For a variable v in a PDF f, the kth order moment of f: ∫−∞∞vkf dv

              • 0th order moment: ∫−∞∞f dv=1
              • 1th order moment: ∫−∞∞vf dv => the expectation
              • 2nd order moment: related to the variance

              In NDF, the 0th order moment of m: Total number concentration

              N(t,x)≡mξ,0≡∫Ωξnξ(ξ) dξ

              Since nξ is the (expected) number unit physical space volume, with all possible ξ (cancel the dependence of ξ)

              An arbitrary moment:

              mξ,k≡∫Ωξξ1k1⋯ξmkm nξ(ξ) dξ

              So different NDFs can be chosen: length, volume, mass (if the material is specified and known)

              Length-Based NDF

              Length base (ξ→L): nL(t,x,L) [1L31L]

              nL(t,x,L) dL: expected # density b/w L & L+dL

              • Total num conc (0th order):

                N≡mL,0=∫0∞nL(L) dL[1L3]

                Mean particle size (one def, the eigenlength)

                L10≡1N∫0∞LnL(L) dL=mL,1mL,0[1/L31/L2=1L]

              • kth order:

                mL,k≡∫0∞LknL(L) dL

                Mean particle size (one def): mL,k+1mL,k

              • Sauter mean diameter: L32=mL,3mL,2

              => can convert to higher dimension (volume density …)

              Assume: V∼L3: V=kVL3 (k - vol shape factor)

              Volume-density function:

              VL(t,x,L) where VL≡kVL3nL

              Volume-fraction density: (normalized VL)

              αV(t,x,L) where αV≡L3nL∫0∞L3nL(L) dL(=VL∫LVL dL)

              Volume-Based NDF

              Volume-based NDF: nV(t,x,V)

              nL(L) dL=nV(V) dV=dV=3kVL2 dLnV(kVL3) 3kVL2 dL

              Convert between nV and nL:

              nL(L)=3kVL2nV(kVL3)mV,k≡∫0∞VknV(V) dV=∫0∞(kVL3)knL(L) dL=kVk mL,3k

              Mass-Based NDF

              Mass-based NDF: nM(t,x,M)

              Consider to connect with length: M=kMLDf (Df is the fractal dimension, 1<Df≤3, for spheric, Df=3)

              nL(L)=DfkMLDf−1 nM(kMLDf)mM,k=kMk mL,Dfk ;mM,1=∫0∞MnM dM(total ptc mass)
